# 🧩 8 Puzzle AI Solver

An AI-powered 8 Puzzle game using the **A* search algorithm** with **Manhattan Distance heuristic**.

## 🔥 Features

- 🎮 Interactive puzzle interface  
- 🔀 Shuffle functionality  
- 🤖 AI Solver using A* algorithm  
- 📊 Move counter & timer  
- 🎬 Animated solution visualization  
- 🌗 Light/Dark mode (Web version)

## 🛠️ Tech Stack

- **Frontend:** HTML, CSS, JavaScript  
- **Backend/Logic:** Python   
- **Algorithm:** A* Search with Manhattan Distance
